# Learn-Haskell
Learning Haskell at SPRINT3R

1. Hello

Compile and Executable
```
$ghc -o hello hello.hs
```

* Run in GHCi
Prelude> :load hello.hs

Prelude> :reload

* Add function 
f x = 2*x + 1

Prelude> f 5
11

* Haskell is type reference 
Prelude> :t 5
5 :: Num a => a

Prelude> :t (+)
(+) :: Num a => a -> a -> a
Num a = a คือ type ในกลุ่มของ number เป็น int, float ก็ได้
a ตัวสุดท้ายคือ output

Prelude> :t (^)
(^) :: (Integral b, Num a) => a -> b -> a

* Tuple
student :: String -> Int -> (String, Int)
student name age = (name, age)

* List
tail [1,2,3,4,5]
[2,3,4,5]

take 2 [1,2,3,4,5]
[1,2]

[1..5]
[1,2,3,4,5]

[1..]  // infinity list

head [1..] //Lazy evaluation
1

[1,2,3,4,5]

init [1,2,3,4,5]

List comprehension
https://en.wikipedia.org/wiki/List_comprehension

[x | x<- [1..5]]
list ของ x 
โดยใน x นำค่ามาจาก list ของ 1 ถึง 5

[x | x<- [1..5]]
[1,2,3,4,5]
